## Description
Projeto de cadastro e usuario com authenticação em [Nest](https://github.com/nestjs/nest) framework TypeScript .

## Installation

```bash
$ git clone git@github.com:finall00/usuarioAPI.git
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
$ npm run test:cov
```
